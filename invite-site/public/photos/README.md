# Wedding Photos

Place your photos in this folder with the following names:

## Required Photos

| Filename | Purpose | Recommended Size |
|---|---|---|
| `hero.jpg` | Full-screen hero background | 1920×1080 (landscape) |
| `couple-portrait.jpg` | Couple portrait for opening | 800×1000 (portrait) |
| `venue.jpg` | Venue section background | 1920×1080 (landscape) |

## Tips
- Use high-quality JPG or WebP images
- Compress images for web (use tinypng.com or squoosh.app)
- Keep each file under 500KB for fast loading
- The site will show elegant placeholders until you add actual photos
